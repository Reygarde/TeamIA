Troubleshooting:

If you encounter the error message "Error: ENOENT: no such file or directory, scandir '/root/website/node_modules/node-sass/vendor'" while running npm install, there are a couple of things you can try to resolve it:

Run npm rebuild node-sass to re-install node-sass 
npm rebuild node-sass
Delete node_modules folder and re run npm install 
npm install

Tips & Tricks: SCSS: We suggest you do not change any scss files from the ./../assets/scss/ovewrite folders because getting new updates will break your SCSS changes if any you have made. We strongly suggest you create a new custom.scss file and use that instead of overwriting any theme's custom scss files.

[NOTE:] If you encounter any issues or have any questions during the setup process, please don't hesitate to reach out to us via email, support center or contact us profile page. We are always here to help and would be glad to assist you in any way we can.
