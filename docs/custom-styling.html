Styling:

We recommend you customize your theme styles using the user style files. You can customize the theme's styles with the following approaches:

  • Gulp based workflow : You can add your own SCSS and override the theme style in the src/scss/user.scss file. To make broader changes to the design of the theme, such as changing the color scheme or font sizes, use src/scss/_user-variables.scss. Any variable from node_modules/bootstrap/scss/variables or src/scss/theme/_variables.scss can be overridden with your own value.

    To remove bootstrap components, update src/scss/_bootstrap.scss file.

    [NOTE]: You need to create _users.scss and _user-variables.scss files and mut need to include on root scss file.

  • If you are not using Gulp based workflow : You can add your own CSS and override the theme style in the dist/assets/css/user.css file.
